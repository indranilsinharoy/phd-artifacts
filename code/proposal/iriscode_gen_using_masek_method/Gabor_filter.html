Notes on Gabor filters¶

References:¶
  1. Recognition of Human Iris Patterns for Biometric Identification, 2003, Libor Masek

A 2-D Gabor filter over spatial domain \((x,y)\) is represented as [1]:

\[G(x,y) = e^{-2\pi i \left[u_0(x - x_0) + v_0(y - y_0)\right]} e^{-\pi \left[ \frac{(x - x_0)^2}{\alpha^2} + \frac{(y - y_0)^2}{\beta^2}\right]}\]

where \((x_0, y_0)\) specify position in the image, \((\alpha, \beta)\) specify the effective width and length, and \((u_0, v_0)\) specify modulation, which has spatial frequency \(\omega_0 = \sqrt{u_0^2 + v_0^2}\)

The above Gabor filter can also be written as a quadrature pair of two odd symmetric and even symmetric 2D Gabor filters as:

\[G_{even}(x,y) = cos \left( 2 \pi \big[u_0(x - x_0) + v_0(y - y_0)\big] \right) e^{-\pi \left[ \frac{(x - x_0)^2}{\alpha^2} + \frac{(y - y_0)^2}{\beta^2}\right]} \]

and

\[G_{odd}(x,y) = -i . sin \left( 2 \pi \big[u_0(x - x_0) + v_0(y - y_0)\big] \right) e^{-\pi \left[ \frac{(x - x_0)^2}{\alpha^2} + \frac{(y - y_0)^2}{\beta^2}\right]} \]

# A rough study
def gabor1d(x, xo, uo, alpha):
    gauss = (1.0/1)*exp(-pi*((x - xo)**2 / alpha**2))
    evensym = cos(2.0*pi*uo*x)
    ioddsym = - 1j*sin(2*pi*uo*x)
    csinu = evensym + ioddsym
    return gauss*csinu, gauss, csinu, evensym, ioddsym

N = 100
x = linspace(0, N-1, N) - N//2

xo = 0
uo = 0.1
alpha = 7

gt, gauss, csinu, evensym, ioddsym = gabor1d(x, xo, uo, alpha)

# Fourier transforms of the components 
EVENSYM = (1.0/N)*fft(evensym)
IODDSYM = (1.0/N)*fft(ioddsym)

GAUSS = (1.0/N)*fft(gauss)
CSINU = (1.0/N)*fft(csinu)

GTr = fft(real(gt))
GTi = fft(imag(gt))

GT = (1.0/N)*fft(gt)

# Frequency index
freq = fftfreq(N)

ax02 = fig0.add_subplot(122)
